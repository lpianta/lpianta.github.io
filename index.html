Mask Detection

During the Covid19 pandemic, many places required a mask to get in. Using Computer Vision and Deep Learning this algorithm can recognize a face and detect if the person is wearing a mask properly.

Best Books of the 20th Century

After using a script to webscrape the site Goodreads to get the data from the first thousand best books of the 20th century, a web app has been build to explore them and do some statistical analysis. The app has been featured on the official Streamlit site.

Algorithm from Scratch

Machine Learning is based on algorithms. There are many libraries with ready-to-use algorithm that you just have to train and use. For a better understanding, I coded a couple of them from scratch, using only basic Python libraries.

SafeDrive

Developed as Capstone Project for Strive School Bootcamp, this python app check driver's drowsiness, head pose and cellphone usage in real time thanks to a combination of Computer Vision and Deep Learning techniques.
